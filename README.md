# GitLab

Manage GitLab projects, merge requests, issues, pipelines, branches, tags, and search via the GitLab REST API v4. Works with GitLab.com and self-hosted instances.

## Categories

- Developer Tools
- Version Control

## Credentials

| Key | Label | Kind |
|-----|-------|------|
| `url` | GitLab URL | text |
| `token` | Personal Access Token | password |

## Installation

This skill is included in [EnterpriseAgentOS](https://officeos.co) and requires no separate installation.
